1.1 Attributes

1.1-1 UnderlyingCategory
‣ UnderlyingCategory( UC )( attribute )

Return the category \(C\) underlying the finite coproduct cocompletion category UC := FiniteCoproductCocompletion( \(C\) )).

1.1-2 YonedaEmbeddingOfUnderlyingCategory

1.4-1 Cocartesian braiding
gap> LoadPackage( "FiniteCocompletion" );
true
gap> LoadPackage( "Algebroids" );

gap> Q := RightQuiver( "Q(A,B)[]" );

gap> FQ := FreeCategory( Q );

gap> Fam := FiniteCoproductCocompletion( FQ );

gap> A := Fam.A;
<A projective object in
 FiniteCoproductCocompletion( FreeCategory( RightQuiver( "Q(A,B)[]" ) ) )>
gap> B := Fam.B;
<A projective object in
 FiniteCoproductCocompletion( FreeCategory( RightQuiver( "Q(A,B)[]" ) ) )>
