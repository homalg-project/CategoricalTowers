1.1 Constructors

1.1-1 DiscreteCategory
‣ DiscreteCategory( filt )( operation )

Returns: a CAP category

Construct a discrete category out of the GAP-filter ValueGlobal(filt).

gap> LoadPackage( "DiscreteCategories" );
true
gap> D := DiscreteCategory( "IsInt" );
DiscreteCategory( "IsInt" )
gap> Display( D );
A CAP category with name DiscreteCategory( "IsInt" ):

12 primitive operations were used to derive 74 operations for this category
which algorithmically
* IsPosetCategory
* IsEquippedWithHomomorphismStructure
and furthermore mathematically
* IsSkeletalCategory
gap> one := 1 / D;
<An object in DiscreteCategory( "IsInt" )>
gap> Display( one );
1
gap> IsWellDefined( one );
true
gap> IsWellDefined( "1" / D );
false
gap> two := 2 / D;
<An object in DiscreteCategory( "IsInt" )>
gap> id_one := IdentityMorphism( one );
<An identity morphism in DiscreteCategory( "IsInt" )>
gap> MorphismDatum( id_one );
fail
gap> IsOne( PreCompose( id_one, id_one ) );
true
gap> IsOne( MorphismInDiscreteCategory( one, one ) );
true
gap> IsWellDefined( MorphismInDiscreteCategory( one, two ) );
false
gap> HomStructure( one, one );
<(⊤)>
gap> HomStructure( one, two );
<(⊥)>
