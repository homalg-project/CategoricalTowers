14.1-1 IsReflexive
‣ IsReflexive( F )( property )

Returns: a boolean

The argument is a (co)presheaf F. The output is true if F is reflexive, i.e., the F-component of the (co)unit of the Isbell adjunction monad is an isomorphism, otherwise the output is false.
