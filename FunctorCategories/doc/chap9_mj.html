9 The category of quivers in a category

9.1 Constructors

9.1-1 CategoryOfQuivers
‣ CategoryOfQuivers( C )( attribute )

Returns: a CAP category

Construct the category of quivers over the category C.

gap> LoadPackage( "FunctorCategories" );
true
gap> Delta2 := SimplicialCategoryTruncatedInDegree( 2 );
FreeCategory( RightQuiver(
  "Delta(C0,C1,C2)[id:C1->C0,s:C0->C1,t:C0->C1,
                   is:C2->C1,it:C2->C1,
                   ps:C1->C2,pt:C1->C2,mu:C1->C2]" ) ) / relations
gap> Size( Delta2 );
31
gap> PSh := PreSheaves( Delta2 );
PreSheaves( FreeCategory( RightQuiver(
  "Delta(C0,C1,C2)[id:C1->C0,s:C0->C1,t:C0->C1,is:C2->C1,it:C2->C1,
                   ps:C1->C2,pt:C1->C2,mu:C1->C2]" ) ) / relations,
SkeletalFinSets )
gap> QuiversInDelta2 := CategoryOfQuivers( Delta2 );
CategoryOfQuivers( FreeCategory(
RightQuiver( "Delta(C0,C1,C2)[id:C1->C0,s:C0->C1,t:C0->C1,
is:C2->C1,it:C2->C1,ps:C1->C2,pt:C1->C2,mu:C1->C2]" ) ) / relations )
gap> datum0 := CoYonedaLemmaOnObjects( PSh.C0 );
[ [ <(C0)>, <(C1)>, <(C2)> ],
  [ [ 1, (C1)-[(id)]->(C0), 0 ], [ 0, (C0)-[(s)]->(C1), 1 ],
    [ 0, (C0)-[(t)]->(C1), 1 ], [ 2, (C2)-[(is)]->(C1), 1 ],
    [ 2, (C2)-[(it)]->(C1), 1 ], [ 1, (C1)-[(ps)]->(C2), 2 ],
    [ 1, (C1)-[(pt)]->(C2), 2 ], [ 1, (C1)-[(mu)]->(C2), 2 ] ] ]
gap> q := CreateQuiverInCategory( QuiversInDelta2, datum0 );
<An object in CategoryOfQuivers( FreeCategory(
 RightQuiver( "Delta(C0,C1,C2)[id:C1->C0,s:C0->C1,t:C0->C1,
 is:C2->C1,it:C2->C1,ps:C1->C2,pt:C1->C2,mu:C1->C2]" ) ) / relations )>
gap> IsWellDefined( q );
true
