12.1-2 UnionOfMultipleDifferences
‣ UnionOfMultipleDifferences( D1, D2, ... )( function )

Returns: an object in a CAP category

If D1=\(A1-B1\), D2=\(A2-B2\), ..., then the output is DirectProduct\((A1,A2,...)\) - Coproduct\((B1,B2,...)\).
