  [33X[0;0YThis constructor returns a map (homomorphism) of finitely presented modules.
  It  is  represented  by the homalg matrix [3Xmat[103X relative to the current set of
  generators  of  the  source  homalg module [3XM[103X and target module [3XN[103X. Unless the
  source  module  is  free  [13Xand[113X given on free generators the returned map will
  cautiously  be  indicated  using parenthesis: [21Xhomomorphism[121X. To verify if the
  result  is indeed a well defined map use [2XIsMorphism[102X ([14Xhomalg: IsMorphism[114X). If
  the  presentations  of the source or/and target module are altered after the
  map  was  constructed,  a  new  adapted  representation matrix of the map is
  automatically  computed  whenever  needed.  For this the internal transition
  matrices  of  the  modules  are  used.  If  source  and target are identical
  objects,  and  only  then,  the  map is created as a selfmap (endomorphism).
  homalg  uses  the so-called [13Xassociative[113X convention for maps. This means that
  maps  of  left  modules  are  applied  from the right, whereas maps of right
  modules from the left.[133X
